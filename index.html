File Deduplicator

Find and remove identical files using SHA-256 fingerprinting. Recover gigabytes of wasted disk space with a single scan.

  • SHA-256 hashing — finds true duplicates regardless of filename
  • Size-first optimisation — blazing fast even on large drives
  • Filter by 20 common file types
  • Move to Recycle Bin, folder, or delete
  • Full after-action report with space recovered
