Privacy Policy

📅 Last updated: 2026-04-11  |  This policy covers both this website and the downloadable software.

1. Who We Are

This website ("WinUtils", "we", "us") provides free, open-source Windows utility software for download. If you have questions about this policy, please open an issue on our GitHub repository.

2. The Downloadable Applications — Zero Data Collection

The applications available for download on this site (File Deduplicator and De-Bloat Pro) collect absolutely no data. Specifically:

  • They contain no network code of any kind. They are physically incapable of making any internet connection.
  • They do not collect, store, or transmit file names, file paths, system information, usage statistics, crash reports, or any other data.
  • They do not create accounts, require sign-in, or contact any remote server.
  • They do not install background services or scheduled tasks.
  • All operations are performed locally on your machine and the results remain exclusively on your machine.

You can verify these statements by reviewing the full source code on GitHub.

For more information on how Google uses data from sites that use its advertising products, please see: How Google uses information from sites or apps that use our services.
