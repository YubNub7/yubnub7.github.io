Get in Touch

Contact WinUtils

Bug reports, feature requests, and general questions are all welcome.

WinUtils is a small open-source project. The fastest way to get a response is GitHub — issues and discussions are monitored regularly. For everything else, email works fine.

🐛

Found something broken? Open an issue on GitHub with your Windows version, the tool version, and what you expected vs. what happened.

Contribute

WinUtils is open source under GPL v3. Pull requests are welcome — see the repository README for contribution guidelines.

View on GitHub
Before reporting a bug: Please check that you're running the latest version of the tool (the version number is shown in the title bar), and that your issue isn't already covered in the File Deduplicator troubleshooting section or De-Bloat Pro troubleshooting section.

Response times

This is a solo project maintained alongside a full-time job. GitHub issues typically get a response within a few days. Email may take up to a week. Urgent security disclosures should be sent by email with "Security" in the subject line.
