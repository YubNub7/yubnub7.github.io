About WinUtils

Built by a developer who needed better tools

Free, open-source Windows utilities with no subscriptions, no telemetry, and no dark patterns.

Why WinUtils exists

Every developer eventually runs into the same problem: you need a simple utility to clean up your PC or find duplicate files, and every tool you find is either paywalled after a free trial, stuffed with adware, or quietly installs a browser toolbar you never asked for. CCleaner used to be the gold standard for PC cleaning — until it got acquired, bloated, and started bundling software. Duplicate file finders are everywhere, but most require payment to actually delete anything.

WinUtils started as a personal project to solve both problems. The goal was straightforward: write utilities that do exactly what they claim, show you everything they plan to do before doing it, never touch the network, and cost nothing. The result is two tools — File Deduplicator and De-Bloat Pro — that have been running on personal machines and shared with colleagues ever since.

What we build

📄

File Deduplicator

Finds genuinely identical files using SHA-256 hashing — not just same-name files, but byte-for-byte identical content regardless of filename or location. Safely removes duplicates to the Recycle Bin, a quarantine folder, or permanently, with a full after-action report.

🛡️

De-Bloat Pro

A transparent alternative to bloated system cleaners. Removes temp files, browser cache, and orphaned registry entries — but only after showing you exactly what it plans to remove. Every registry operation is backed up to a .reg file before anything is touched.

Our principles

1
No network code. Both tools contain zero network imports. They cannot phone home, check for updates automatically, send crash reports, or contact any server. This is verifiable in the source code on GitHub — not a policy statement, a technical fact.
2
Preview before action. Every destructive operation — file deletion, cache removal, registry cleanup — is shown to you before it happens. Analyze mode is always available and always free.
3
Backups before registry changes. De-Bloat Pro exports a .reg backup file before removing any registry key. If something goes wrong, you can restore with a double-click.
4
Open source, always. All code is published under the GNU GPL v3.0. You can read it, compile it yourself, fork it, or audit it. There are no hidden builds or binary-only releases.
5
Free, permanently. Not "free tier with paid upgrade." Not "free for 30 days." Free. There is no premium version. All features are available from the first launch with no account required.

Who maintains this

WinUtils is maintained by a software developer with a background in network engineering and systems administration. The tools grew out of real frustration with the available alternatives — and a belief that utilities this straightforward shouldn't cost money or come with strings attached.

Both tools are actively maintained. Bug reports and feature suggestions are welcome via GitHub or the contact page.

Want to contribute? WinUtils is open source under GPL v3. Pull requests, issue reports, and documentation improvements are all welcome on GitHub.
